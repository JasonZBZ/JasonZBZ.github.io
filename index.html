如何搭建一个hexo博客

Mr. X

发布于：Mar 14, 2020

如何搭建一个hexo博客

第一部分：

首先下载nodejs，在官网下载即可（一直next，建议安装在默认目录）

然后这里需要下载git（https://blog.csdn.net/sanxd/article/details/82624127）

然后打开git bash

先需要下载一个镜像：

npm install -g cnpm –registry=https://registry.npm.taobao.org

然后安装hexo

cnpm install -g hexo-cli

hexo -v验证一下

新建一个文件夹

mkdir blog（出错可以干掉重来）

然后进入blog目录下

1.初始化

hexo init

2.启动博客

hexo s

3.新建一篇文章

hexo n “xxxxx”

hexo clean

hexo g(generate)

hexo s

这样你的博客就搭建起来了！

如果需要修改则可以进入 blog/source/_posts/目录下

对文件进行修改，也可以用能打开markdown文件的编辑器打开（这里安利typora!很好用的md编辑器）

第二部分：如何将博客部署到github服务器上

1.打开github，创建一个新的repository

注意这里的name必须是Owner.github.com.io(Owner为你的名字)

2.下载一个git插件

cnpm install –save hexo-deployer-git

3.设置配置文件

vim _config.yml

注意这里type: repo:后面必须有一个空格，

repo则是你自己仓库地址

4.部署

hexo d

然后输入github账号和密码

3.更换主题：

推荐github.com/litten/hexo-theme-yilia

进入blog目录

这里不推荐直接下载，可以先部署到码云上，再下载速度比较快

https://blog.csdn.net/xy_zhangchn/article/details/81558144?depth_1-utm_source=distribute.pc_relevant.none-task&utm_source=distribute.pc_relevant.none-task（这是一个博主写的，亲测有效）

下载成功后

vim _config.yml

修改theme即可

最后再hexo d就行（在github不一定立马生效，需要多刷新几遍）

啊，到这里基本的操作就完成了，至于后续的美化可以去网上学学其他大佬的操作。

最后总结一下我搭建时碰到的几个问题：

1.安装完cnpm后，执行时cnpm不是内部命令:

找到安装目录

把该路径添加到系统环境变量path即可

2.hexo init报错

WARN Failed to install dependencies.Please run ‘npm install’ manually!

这里直接cnpm install即可

3.若缺少什么插件，直接安装即可

hexo s 错误:ERROR Plugin load failed: hexo-renderer-pug

cnpm install hexo-renderer-pug –save即可

4.在git clone时遇到Authentication failed 和Incorrect username or password ( access token )的问题

https://blog.csdn.net/ezconn/article/details/81872785（解决方法）

大多数问题，都能在网上找到答案，希望大家顺利搭建成功！
